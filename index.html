Web Editor RIE KIRAMEKI

東京都在住のWeb編集者，きらめきりえです。オンラインスクールでWeb制作について学び、出版業界の紙編集者からWeb編集者へと転職活動中です。これからさらにWeb制作の技術や知識を身につけて、Webメディアや企業マーケティングのためのコンテンツ作りができる編集者になりたいと思っています。

Skills

Web

HTML/CSSでのコーディングができます。JavaScript、PHPも現在勉強中です。

Editing

7年間編集者として出版社で勤務し、編集校正の技能はもちろん、企画立案検証のスキルを身に着けました。小中学生向けの教材編集が最も得意です。

Chinese
